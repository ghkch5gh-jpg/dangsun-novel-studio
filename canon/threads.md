# 떡밥 (열림/회수) — 런닝, 매 화 전체 갱신

## 열림
- **[[단우건축사사무소]]의 실적 벽 / 대표사 출전(핵·잔혹함).** ◇◇시립미술관은 '최근 5년 3,000㎡ 이상 2건' 요구, 단우 1건뿐. ▽▽동에서 한 줄 더 벌어야 미술관을 낸다. 미회수.
- **징검다리 판 — ▽▽동 생활문화센터(핵·진행).** 3,400㎡, 단우 첫 대표사 단독 출전. 열린 ㄷ자 배치+처마·마루(대청)+뿌리 보호구역 무손상까지 진행. 미회수.
- **정성(무대·그림) vs 정량/생태 정당성(전장 구조·핵).** STX가 나무를 '무대'로, 단우는 '마루'로. 37화 반전: 생태 카드(무대=고사)를 STX가 무접지 부유 데크+수목 자문으로 선제 차단—'예쁜 쪽이 옳기까지' 함. 생태 정당성 축이 무력화됨. 이제 마루가 이길 다른 힘(주민·운영·명분)이 관문. 미회수.
- **'이번엔 진다' 복선(아크 방향).** 명분은 옳으나 조용한 안이 STX 물량·무대에 밀릴 수 있음. 37화에서 가을의 유일 카드(생태)가 선제 차단되며 그림자 짙어짐—전승 깨기. 미회수.
- **가을의 '봄'의 한계(양식).** '옳게 봄'이 곧 '통과'가 아님. 37화: 급소를 한발 먼저 봤으나 상대가 이미 막아 둠—본 것이 무력해지는 새 형태의 벽. 미회수.
- **그린 손 vs 말하는 입 / 이름(3막 핵).** ▽▽동은 대표사 출전—이름까지 남기는 첫 시험대. 37화 엄마 문자로 크레딧 상실의 사적 아픔 활성. 미회수.
- **STX와의 재대결 / [[차도경]](정면·행동).** 무대안 서른 장 티저+무접지 부유 데크 상세+수목 보호 기술사 자문으로 생태 결격 선제 차단. 양측 다 대표사. 미회수.
- **[[남기준]]/JW와의 관계(균열·처리됨→여진).** 슬래브 실시상세 검토 SOS를 가을이 수락(봐주기로 결정)—거부 시 15% 위태·제 슬래브 선이 죽는 것 회피. 크레딧은 여전히 흡수, 검토 의견서로 손만 종이에 남김. 흡수 구조의 여진 유지. 미회수.
- **발주처의 눈 / 증인(전진).** □□구 시설직 위원의 ◇◇시립미술관 자문 위촉 확인(이름까지). 다음 판 무기이자 변수. 미회수.
- **회사 인력 / [[배수경]](인물·진행).** 37화 냉소를 실무 증언(조명 데크 준공 반려 경험)으로 뒤집어 처음 가을 옆에 앉아 협업 반 발—미화해에서 전진. 단 'STX면 자문 붙였을 것' 선제 경고로 냉소의 통찰도 유지. 어린이도서관 실시 크로스체크 세 번째 눈. 미회수.
- **크레딧 소거 / 실시 하청 축소(핵·진행).** 배분표 하청화 유지. 37화: 남기준 슬래브 SOS를 가을이 봐주되 크레딧은 흡수, 검토 의견서로만 손을 남기는 절충. 회수 궤도 미착.
- **어린이도서관 실시설계 — 안정화(진행).** 배수경 보강, 막내 2인 크로스체크. 유지.
- **[[정우람]]의 자리 / 멘토(유지).** 37화: 슬래브 SOS의 '봐줘도 손해·안 봐줘도 손해' 딜레마를 짚어 가을의 결정에 프레임을 줌. 멘토 유지.
- **가을의 관계 stake(3막 연료).** 엄마 문자(□□구 기사에 JW만)—크레딧 상실을 가족에게 설명 못 하는 사적 아픔으로 활성. 미회수.

## 회수
- (37화) **남기준 슬래브 SOS 처리(봐주기 결정·의견서 절충) / STX 무대안 생태 급소 읽기 + 배수경 실무 증언 협업 / 차도경의 무접지 부유 데크 선제 차단으로 생태 카드 무력화('예쁜 쪽이 옳기까지').**
- (36화) 빈 중앙 명분 첫 회수 / 답사·행간으로 '지붕 없는 사랑방' 발견 / 소장 손이 처마·마루 선 완성 / STX '무대'안 행동 노출 / 배수경 냉소 vs 이상주의 첫 충돌.
- (35화) ▽▽동 '포위=고사' 오독 교정 / 열린 ㄷ자 첫 배치 / 대표사 착수·위계 없는 첫 합의 / STX 대표사 참가·배수경 합류·미술관 시설직 위원 자문.
- (34화) JW 하청 축소 구체화 / 실적 벽 못박힘 / 보호수 첫 읽기. (33화) □□구 당선·부분개방 카드·차도경 벽 복원·실적 한 줄·시설직 위원 기억. (32화) 옳은 책상 첫 실행·38% 최종 방어·고민석의 손. (31화) 실시 크로스체크 첫 성과. (29화) '말하는 입' 첫 발동. (20화) 어린이도서관 당선·[[정우람]] 잔류. (14화) 소장-실장 마당 정산. (13화) 합본 잠금·제출.
