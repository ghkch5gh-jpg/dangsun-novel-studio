# dangsun-novel-studio

현상설계 오피스 드라마 자동연재 「당선까지 한걸음」. dangsun.kr/novel/studio.

1등 아니면 0원의 설계공모 토너먼트. 갓 입사한 신입 설계자 **서가을**이 작은 현상 아틀리에 '단우건축'에서, 남들이 못 보는 '심사의 언어'를 읽는 눈을 무기로 첫 공모에 던져진다. 하지만 옳은 답을 *통과시키는 것*은 또 다른 게임이다 — 위계·마감·정치 앞에서. (시즌1 100화)

구조는 14_웹소설(헌터)·15_무협·16_로판과 동일: 락드 canon(연표=고정 전사·설정) + 연속성/재미 채점 + 편집자 리뷰 자가발전 + 개입(novel_id=studio). 매일 08:45 KST.

## 한 끗
- 천재 무쌍이 아니다. 가을의 우위는 "정답을 안다"가 아니라 **"문제를 정확히 본다"**.
- 보는 것과 이기는 것 사이엔 늘 사람·마감·정치가 있다 → *직관 vs 통과*의 드라마.
- 무기: 심사의 언어를 읽는 눈 + AI·파라메트릭(라이노·그래스호퍼) 네이티브.

## 구조
- `canon/` — 🔒 락드: `premise.md`(한 끗) · `arc.md`(100화 아크) · `world.md`(현상설계 업계·사무소 세계관) · `timeline.md`(고정 연표·전사) · `characters/*.md`(인물 바이블). 런닝: `threads.md`.
- `state.md` `synopsis.md` — 런닝 상태/시놉시스(매 화 갱신).
- `scripts/build-local.mjs` — 생성기(생성→연속성 체크→재미 채점→문체 린트→1회 재생성).
- `scripts/review.mjs` — 편집자 리뷰(자가발전 루프).
- `scripts/run-daily.ps1` — 매일 08:45 작업 스케줄러 진입점.

## 인물
서가을(신입·주인공) · 고민석(소장, 손맛의 대가) · 정우람(실장·사수, 번아웃) · 장세훈(동기·라이벌) · 민경애(차장, 살림꾼) · 차도경(경쟁사 STX 에이스).
