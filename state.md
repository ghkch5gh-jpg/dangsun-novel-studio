# 현재 상태 (state.md) — 런닝, 매 화 전체 갱신

- **회차**: 39 (3막 — ▽▽동 생활문화센터 미니아크 / 두 판 동시)
- **장소**: 단우건축사사무소 모형 작업대·회의 테이블
- **시각/계절**: 11월 말~12월 초, 낮에서 밤. 중간 크리틱 직전이며 □□구 실시·어린이도서관 실시·▽▽동 공모가 중첩.
- **소지/상황**: **(1) 손 모형 사고 발생.** 1:200 대지 모형에서 1mm 회색 보드 14번을 중복 절단하고 15번을 누락해 실제 표고 200mm가 사라짐. 이미 접착한 상부 열세 겹을 뜯어 재제작하면서 최소 두 시간 추가 손실. **(2) BF 함정 발견·1차 우회.** 거짓 표고 때문에 남서 주출입 접근로가 실제 고저차 600mm·길이 5.4m의 1:9 경사임이 가려져 있었음. 가을이 경사 1:12 이하·보호구역 침범 0 조건으로 검산해 대지 경계를 따르는 7.8m 우회로를 잡았고, 행사 관람석 8개를 덜어 공간을 확보함. 최종 상세 검산은 남음. **(3) ‘삼백오십구 일의 마루’ 구조적 약점 표면화.** 다목적실 접이문을 계속 연 장면으로는 겨울 열손실과 저녁 강좌 소음을 설명할 수 없고, 닫으면 마루와 프로그램이 끊김. **(4) 장세훈의 독자 대안.** 다목적실을 한 모듈 안쪽으로 밀고 폭 2.4m 유리 완충 공간을 두는 평면을 제안해 겨울·소음 대응은 개선했으나, 작은도서관이 지침 최소면적보다 24㎡ 부족해 그대로 채택 불가. **(5) 고민석의 책임 유지.** 패널 중앙 두 장의 ‘삼백오십구 일’ 운영축을 철회하지 않고, 자신이 선택한 중심 논리를 계절 단면과 실제 평면으로 책임지기로 함. **(6) 미해결 피난 결함.** 폭 900mm 접이문 수납 포켓을 적용하면 문짝 일곱 장이 완전 개방 상태에서 다목적실 두 번째 피난구를 가림. 마루가 무대로 열리는 순간 피난이 막히는 물리적 모순이 1:50 종이 모형에서 드러남.
- **신체/심리**: JW 상세 검토로 잃은 외주 슬롯에 이어 손 모형 재제작까지 겹쳐 피로와 일정 압박이 심화. 가을은 모형의 거짓 표고를 먼저 읽어 배치 결격을 피했지만, 자신의 중심 개념이 겨울·소음·피난을 동시에 통과하지 못한다는 책임을 체감. 세훈의 정석 대안을 경쟁이 아니라 실제 선택지로 인정하기 시작함.
- **즉시 목표**: (1) 접이문 완전 개방 시에도 두 방향 피난을 확보할 수납 위치·문 분할·다목적실 배치를 찾기. (2) 겨울 열손실과 강좌 소음을 줄이되 작은도서관 최소면적과 마루의 직접 관계를 지키기. (3) 수정한 7.8m BF 접근로의 횡경사·휴게참·보호구역 이격을 최종 검산. (4) 훼손된 대지 모형 상부 열세 겹을 복구해 중간 크리틱 일정 손실을 줄이기. (5) ‘6일/359일’ 운영표를 계절별 개방 상태와 일치시키기.
- **장기 목표**: 자기 이름으로 안을 책임지는 설계자. ▽▽동 당선으로 3,000㎡ 이상 실적 두 번째 줄을 확보해 ◇◇시립미술관 대표사 출전 자격을 만들고, 보는 눈뿐 아니라 평면·법규·마감·조직을 함께 통과시키는 힘을 갖추기.

40화: 접이문 수납부가 두 번째 피난구를 가리는 결함을 두고 가을의 직접 개방안과 세훈의 완충 평면이 다시 맞붙는다. 문 분할, 피난거리, 방풍·차음, 본실 최소면적을 동시에 검산해야 하며, 중간 크리틱 시간은 손 모형 복구 때문에 이미 줄었다. ‘삼백오십구 일의 마루’를 유지하되 무엇을 덜어낼지 선택하게 만들고, STX의 물량·무대 우위와 단우 안의 미완성 상태를 함께 밀어 당선이 보장되지 않는 흐름을 유지한다.
